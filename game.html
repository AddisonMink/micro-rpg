  • Backspace/Delete - Cancel
  • Tab - Info
  • 1/2 - Numeric inputs.

Readme

This is demo of an RPG I'm making. So far it consist of a single battle.

The framework is a standard row-based JRPG battle system with a couple of twists.

  • The front and back row act as range bands. Every action has a range which determines which row it can be used from and which rows it can target.

  • All special actions are attached to limited-use items. A character's magic level determines which special actions it can use from each item.

You control a Magician and a Galoot, and you are fighting 3 Scamps. The Magician has high magic, so he can use all item actions. The Galoot has low magic so he can only use some item actions. The Galoot is armored, so all damage against him is reduced by 1.
